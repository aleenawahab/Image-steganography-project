# Image Steganography Project

This project hides a secret image inside a cover image using LSB technique.

## Features
- Hide image inside another image
- Extract hidden image

## Technologies
- Python
- OpenCV
- NumPy

## How to Run
1. Run main.py
2. Hidden image will be generated
3. Extracted image will be saved
